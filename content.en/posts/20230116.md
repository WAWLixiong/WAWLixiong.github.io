## golang

string是utf8编码的，底层存储的是1-4个字节

```golang
s := "hello"
fmt.Println(len(s)) // 5
s := "汉"
fmt.Println(len(s)) // 3
```

string的正确遍历方式

```golang
s := "hêllo"
for i := range s {
    fmt.Printf("position %d:%c\n", i, s[i])
}
fmt.Printf("len=%d\n", len(s))
```
